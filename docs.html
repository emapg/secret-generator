Auth Secret Generator API

Endpoints

/generate-secret

Generates a random authentication secret.

Methods: GET, POST

Query/Body Parameters:

  • length (optional): Length of the secret (default is 32).
  • algorithm (optional): Algorithm for encoding (default is 'hex'). Options: 'hex', 'base64'.
  • charset (optional): Character set to use. Options: 'alphanumeric', 'numeric', default is 'alphanumeric'.

/validate-secret

Validates a given secret.

Query Parameters:

  • secret (required): The secret to validate.
  • length (optional): Expected length of the secret.
  • charset (optional): Character set to validate against. Options: 'alphanumeric', 'numeric'.

Examples

GET /generate-secret

GET /generate-secret?length=64&algorithm=base64

GET /validate-secret?secret=yourSecretHere&length=32&charset=alphanumeric
